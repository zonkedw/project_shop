# FoodShop - Интернет-магазин продуктов питания

Современный и полнофункциональный интернет-магазин продуктов питания, созданный на React.

## Особенности

- **Адаптивный дизайн** - работает на всех устройствах
- **Функциональная корзина** - добавление, удаление, изменение количества товаров
- **Современный UI** - красивый и интуитивно понятный интерфейс
- **Управление состоянием** - с использованием React Context API
- **Категории товаров** - акции, новинки, популярные товары
- **Поиск товаров** - быстрый поиск по каталогу
- **Рейтинги и отзывы** - система оценок для товаров

## Быстрый старт

### Установка зависимостей
```bash
npm install
```

### Запуск приложения
```bash
npm start
```

Приложение откроется в браузере по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена
```bash
npm run build
```

## Структура проекта

```
src/
├── components/          # React компоненты
│   ├── Header/         # Шапка сайта
│   ├── Banner/         # Баннер с акциями

```

## Технологии

- **React 18** - основной фреймворк
- **React Context API** - управление состоянием
- **Lucide React** - иконки
- **CSS3** - стилизация
- **Responsive Design** - адаптивность

## Основные компоненты

### Header
- Логотип и навигация
- Поиск товаров
- Иконка корзины с счетчиком
- Контактная информация

### ProductCard
- Изображение товара
- Название и описание
- Цена (обычная и со скидкой)
- Рейтинг и отзывы
- Кнопка "В корзину"

### Cart
- Список товаров в корзине
- Изменение количества
- Удаление товаров
- Общая сумма заказа
- Кнопка оформления заказа

## Кастомизация

### Изменение товаров
Отредактируйте файл `src/data/products.js` для добавления своих товаров:

```javascript
{
  id: 1,
  name: "Название товара",
  price: 99.90,
  originalPrice: 119.90, // опционально для скидки
  image: "ссылка_на_изображение",
  rating: 4.5,
  reviews: 128,
  discount: 18 // процент скидки
}
```

### Изменение цветовой схемы
Основные цвета определены в CSS переменных:
- `#ff6b35` - основной оранжевый цвет
- `#28a745` - зеленый цвет (кнопки, навигация)
- `#333` - темный текст
- `#666` - серый текст
